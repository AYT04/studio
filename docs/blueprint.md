# **App Name**: TubeGuide

## Core Features:

- Live TV Guide UI: Display YouTube channels and playlists in a scrollable, Live TV Guide-like interface.
- Instant Playback: Enable instant playback of selected YouTube content within the app.
- In-App Search: Implement a search functionality for channels and videos available on YouTube.
- Favorites System: Allow users to mark channels and videos as favorites, creating a personalized guide.

## Style Guidelines:

- Background: Dark charcoal (#333333) to reduce eye strain.
- Primary: Muted blue (#6495ED) for channel and content highlights.
- Accent: Bright white (#FFFFFF) for text and key interactive elements.
- Font pairing: 'Inter' sans-serif for body text and 'Space Grotesk' sans-serif for headings to provide a modern yet readable experience.
- Use simple, outlined icons in white for navigation and actions.
- Design a grid-based layout to accommodate various YouTube content types.
- Subtle animations when transitioning between the guide and playback screens.
